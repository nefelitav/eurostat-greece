# Eurostat Data for Greece

This project provides an easy-to-use dashboard with key Eurostat data for Greece. 
It is designed to give me quick and direct access to important economic, demographic, environmental, and political indicators. 
By centralizing this information, I can easily track and evaluate how my country is performing across various dimensions, 
helping me make informed judgments about Greece’s overall progress and challenges.
